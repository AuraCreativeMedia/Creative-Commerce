*  Creative Agent
*  Creative Conversion
*  Creative Dual Engine - 1.3.5
*  Creative Publicity
*  Creative Stock Maximiser
*  Creative Time Saver
*  Creative Trade Booster

(Plugins will be prefixed with Aura due to brand change)


== Description ==

The Creative Commerce bundle is a suite of plugins which enable a variety of additional functions in Wordpress & WooCommerce.



== Installation ==

1. Upload the plugins to a fresh WordPress/WooCommerce website
2. Activate the plugins through the 'Plugins' menu in WordPress
3. Check the STATUS screen for required dependencies
