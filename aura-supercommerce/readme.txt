The Creative Commerce is a foundational plugin that underpins the following:

*  Creative Agent
*  Creative Conversion
*  Creative Dual Engine
*  Creative Publicity
*  Creative Stock Maximiser
*  Creative Time Saver
*  Creative Trade Booster

(Plugins may be prefixed with "Aura")


== Description ==

The Creative Commerce Foundation enables a variety of additional functions in Wordpress & WooCommerce.


== Installation ==

1. Upload the plugins to a fresh WordPress/WooCommerce website
2. Activate the plugins through the 'Plugins' menu in WordPress
3. Check the STATUS screen for required dependencies
